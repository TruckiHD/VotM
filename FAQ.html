What Is VOTM ?

Village of the moon is a real-time strategy game inspired by Werewolf. Build your village and defend it from the evil creatures.

How Do I Get Access To The Game ?

Since the game is far from being finished, you can currently only test it as a game tester. To become one, please check out our offical Discord server to apply or the contact page. Otherwise, the game will be released on Steam in a year or two if all goes well!

Is Village Of The Moon Available On Mac Or Linux ?

Currently we do not plan the game available on Mac and Linux. If we eventually get around to it later, we will try to make it available for Mac and Linux users as well.

How Can I Submit Bugs Or Crashes ?

You can send bug/crash reports by emailing villageofthemoon@gmail.com. When submitting a bug report, please include the following:

- A detailed explanation of your bug/crash
- Steps to reproduce the bug/crash (if possible)
- A DXDIAG file that contains detailed information about your hardware Task Bar > DXDIAG > Save Information

You can also create a ticket on our offical Discord server, by text our Modmail-Bot.

What are VOTM's PC requirements ?

Since we don't have enough data yet and are always improving the game performance technically, we haven't set a minimum requirement yet. Just try it and if it doesn't work, it's best to wait for new updates, because as already mentioned, they always bring out new performance patches.
