Village Of The Moon

Village Of The Moon is a building strategy game inspired by Werewolf. Do what you've always wanted to do.

Goal

Our goal is to create an RTS game that incorporates elements of Werewolf. You build up your village and have to free it from the werewolves.

Development

When the game is officially released on Steam, it will still be in early access. You can currently apply on Discord to be a game tester and help us improve the experience for all players.

Community

We will be working closely with the community during the development period to get the best out of this project. If you want to support us, feel free to contact us at the contact page.

Our Features

The core features of Village Of The Moon

Camera System

The camera system allows you to view the game world from any perspective. With the automatic Depth Of Field system, beautiful images can be created.

Werewolf Game Characters

Since this game is based on Werewolf, we will try to include as many characters and items as possible in the game.

Day/Night System

With the day and night system, you need to prepare especially for the nights, because as you may already know, most things happen at night in Werewolf.
